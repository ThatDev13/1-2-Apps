# 1-2-Apps 🚀

Eine moderne, multifunktionale **Terminal User Interface (TUI)** App, entwickelt mit Python und [Textual](https://github.com/Textualize/textual). 

**1-2-Apps** vereint drei Produktivitäts-Tools in einer einzigen, leichtgewichtigen Terminal-Anwendung.

---

## ✨ Features

### 📝 Sticky Notes
Verwalte deine Aufgaben mit farblicher Priorisierung direkt im Terminal:
- **Hoch (Rot)**: Sofortige Erledigung erforderlich.
- **Mittel (Gelb)**: Wichtig für den aktuellen Workflow.
- **Niedrig (Grün)**: Backlog oder Ideen.

### ⏲️ Pomodoro Timer
Fokussiertes Arbeiten leicht gemacht:
- Standardmäßig **25 Minuten** Fokus-Zeit.
- Intuitive Steuerung: Start, Stop und Reset.
- Visuelle Darstellung über ein modernes Digits-Display.

### 🤖 Terminal Agent
Integriere die Power von KI in deinen Workflow:
- Unterstützung für **Claude (Anthropic)**, **ChatGPT (OpenAI)** und **Gemini (Google)**.
- Sicherer API-Key-Input innerhalb der TUI.
- Protokollierung der Anfragen und Antworten in einem interaktiven Log.

### 📖 Integrierte Dokumentation
Keine externe Webseite nötig – alle Anleitungen und Tastenkürzel sind direkt im Reiter **Hilfe** innerhalb der App verfügbar.

## 🚀 Installation

Stelle sicher, dass du Python 3.8+ installiert hast.

1. **Repository klonen:**
   ```bash
   git clone https://github.com/ThatDev13/1-2-Apps.git
   cd 1-2-Apps
   ```

2. **Abhängigkeiten installieren:**
   ```bash
   pip install textual openai anthropic google-generativeai
   ```

3. **App starten:**
   ```bash
   python one_two_apps.py
   ```

---

## ⌨️ Bedienung

- **Tabs**: Nutze die Maus oder die Tab-Navigation oben, um zwischen den Tools zu wechseln.
- **Navigation**: Verwende die `Tab`-Taste, um zwischen Eingabefeldern und Buttons zu springen.
- **Beenden**: `Strg + C` schließt die Anwendung sicher.
